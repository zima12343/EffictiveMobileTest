В приложении, для хранения данных используется база данных SQLite.
Никакие дополнительные настройни не требуются. 

Описание API
[POST]
api/Datagenerator -- генерирует новые районы и заказы в них
        (int) areasAmount -- кол-во районов
        (int) amountInArea -- кол-во заказов в каждом райне

[GET]
api/Logger -- возвращает текстовый файл с логами

[GET]
api/Orders/AllOrders -- возвращает текстовый файл со всеми заказами

[GET]
api/Orders/FilterOrders -- возвращает текстовый файл с отфильтрованными заказами
        (GUID) areaId -- id р-на для поиска
        (DateTime) dateTimeOrder -- дата первого заказа (в формате 2024-10-26T21:28:41)
